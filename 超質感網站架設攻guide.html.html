網站架設攻略

從零開始，免費建立你的專屬網站
不需要任何程式背景

目錄

  • 01註冊 GitHub 帳號
  • 02建立 Repository（儲存庫）
  • 03上傳網站檔案
  • 04開啟 GitHub Pages
  • 05取得並分享網址
01
Step 1
註冊 GitHub 帳號

前往 GitHub 官網進行免費註冊。

  • 打開瀏覽器，前往：
    https://github.com/signup
  • 輸入你的 Email 地址
  • 設定 Username（帳號名稱，之後會出現在網址中）
  • 選擇國家／地區（Taiwan）
  • 點選 Create account
  • 完成驗證拼圖（Visual puzzle）
  • 到信箱確認驗證信
注意：Username 會出現在你的網站網址中，建議使用簡短好記的英文名稱。
02
Step 2
建立 Repository（儲存庫）

Repository（簡稱 repo）就是存放網站檔案的地方，類似一個資料夾。

  • 登入後，在首頁找到 Create repository（綠色按鈕）並點選
  • 在 Repository name 欄位輸入名稱，例如：
    my-website
  • Visibility 選擇 Public（公開，GitHub Pages 免費版必須公開）
  • 其他選項不需要更動
  • 往下滑，點選 Create repository 完成建立
重要：Visibility 一定要選 Public，否則 GitHub Pages 免費版無法運作。
03
Step 3
上傳網站檔案

將你的 HTML 檔案上傳到剛剛建立的 repo。

  • 建立完 repo 後，往下滑找到 Quick setup 藍色框框
  • 點選藍色連結 uploading an existing file
  • 點選 choose your files
  • 從你的裝置中選擇所有網站 HTML 檔案
  • 等待上傳完成（檔案名稱會出現在畫面上）
  • 往下滑，點選 Commit changes 儲存
小技巧：如果你有主頁，建議將它命名為 index.html，這樣網址不需要加檔名就可以直接開啟。
04
Step 4
開啟 GitHub Pages

上傳完成後，需要手動開啟 GitHub Pages 功能，才能讓網站上線。

  • 在 repo 頁面，點上方的 More 標籤
  • 選擇 Settings
  • 在左側選單往下滑，找到並點選 Pages
  • 在 Branch 區域，點 None 下拉選單，改選 main
  • 旁邊資料夾選項保持 / (root) 不變
  • 點選 Save
  • 畫面出現「GitHub Pages source saved.」即代表設定成功
等待時間：儲存後約需 1–2 分鐘，GitHub 才會完成網站部署。
05
Done!
取得並分享網址

稍等片刻後，回到 Pages 設定頁面，就會看到你的網站網址。

  • 在 Pages 頁面頂部會出現：
    「Your site is live at...」
  • 你的網址格式為：
    https://你的帳號.github.io/repo名稱/
  • 點選 Visit site 確認網站正常運作
  • 如有多個 HTML 檔案，在網址後加上檔名即可開啟：
    https://你的帳號.github.io/repo名稱/檔名.html
  • 將網址分享給學生或客戶
🎉 完成！你的網站現在完全免費運作，不需擔心流量限制，隨時都可以分享給別人。
